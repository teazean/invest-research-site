**一句话结论：** 紫金矿业已经验证了“找矿/买矿 -> 建设和技改 -> 产量 -> 现金流”的自身执行结果；但缺少同类型竞对的成本、工期、资源方选择和ROIC对照，因此只能保留“竞争优势假设”，不能从经营增长直接推出相对壁垒。股价回撤的机械桥显示TTM EPS增长、TTM PE大幅收缩；能排除“7月业绩预告触发暴跌”，也能确认6月23日有板块共跌，但无法把整段回撤精确归因给金价、利率、卡莫阿、并购或资金中的某一个。

## 1. 一句话看懂公司

- **公司做什么：** 在全球找矿、买矿、建矿和运营矿山，生产黄金、铜、锌、银、锂、钼；同时做冶炼、贸易和销售。
- **卖给谁：** 黄金交易所和黄金市场、冶炼厂、工业企业、贸易客户。2025年约63%营业收入来自中国大陆客户，其中约27%来自上海黄金交易所。
- **谁付钱：** 下游按黄金、铜等市场价格购买矿产品或冶炼产品。
- **客户为什么买：** 产品达到品位、纯度、交割和合规标准，并能稳定交付。客户通常不会为矿企品牌长期支付很高溢价。
- **主要利润来源：** 自有矿山的“金属售价减单位开采成本”，不是收入看起来很大的冶炼和贸易。2025年剔除冶炼后的矿产品毛利率61.56%，冶炼铜毛利率仅2.08%。
- **最大误区：** 紫金收入不等于矿山价值。低毛利冶炼和贸易会做大收入，真正要跟踪的是权益矿产量、实现售价、单位成本、矿山寿命和资本开支。

|---|---|---|---|---|---|---|---|

### 5.2 为什么“资源多”不能直接推出护城河

资源量只有经过经济可采性、许可证、融资、建设、达产和成本验证，才会变成现金流。高金价还会降低边际品位，使“资源量”随价格假设增加；如果公司高价收购后资本回报不足，资源越多反而占用越多资金。因此本报告把**产量、单位成本、自由现金流和ROIC**放在资源吨数之后。

### 5.3 当前优势的可持续条件

1. 自主勘探和技术经济再评价持续补充资源，降低完全依赖并购的成本。
2. 巨龙、塞尔维亚、卡莫阿和黄金新项目按预算达产。
3. 矿山单位成本涨幅低于金铜价格和同行。
4. 海外本地化、社区、安全与环保体系跟上资产扩张。
5. 并购后自由现金流回报高于融资成本，并保持可控净债务。

“等待”不是看空公司，也不是确定性买卖建议。它表示现有证据足以把紫金列入高优先级观察，但还不应把H1简单年化利润当成永久利润直接下结论。

### 12.1 购买候选资格改变所需证据

旧版曾使用缺少公司指引、历史分布、同行比较或资本成本推导的阈值，现已删除，不能把作者拍定的数字包装成“可执行条件”。资格改变需要以下证据链完整，而不是碰到一个价格数字：

1. **经营质量：** 正式半年报披露收入、矿产品毛利率、经营现金流、单位成本和资产负债表，并能解释利润增长中量、价、成本和并表各贡献；现金流不能显著背离扣非利润。
2. **产量兑现：** 后续季度产量与公司最新正式年度指引比较，写出达成率、项目爬坡和卡莫阿恢复；不再自行发明季度吨数门槛。
3. **资本与摊薄：** 取得Allied、赤峰等交易的实际支付、交割后净债务、利息覆盖、最大充分摊薄股数和少数股东损益，证明增长没有被融资成本和摊薄吃掉。
4. **正常化利润：** 用3-5年中周期实现售价、权益产量、单位成本、税费、少数股东损益和充分摊薄股数逐项重建；缺任何关键项时保持“未可靠计算”。
5. **合理倍数：** 从增长持续期、增量ROIC、现金转化、分红、净稀释、国别与治理风险和要求回报率推导，并与同口径历史/竞对范围交叉核验；不直接套用任意整数倍数或同行平均。
6. **安全治理：** 没有重大安全、环保、社区事故或迟报瞒报；新班子资本配置和信息披露经至少一个完整报告期验证。

只有经营证据、正常化EPS和合理倍数三者同时完成，才能重新评估“具备购买候选资格”；这仍不是自动交易指令。用户若要决定买入、持有、加仓或退出，需补充持仓状态、真实计划持有期限和风险承受能力，再交给 `investment-decision-discipline`。

## 15. 小白术语表

| 词 | 白话解释 | 为什么重要 |
|---|---|---|
| 矿石流五环归一 | 把地质勘探、采矿、选矿、冶炼和环保/生态作为一个系统共同优化，而不是各部门只优化自己一段 | 可能降低边界品位、回收率、运输和建设的总成本；但相对竞对好多少仍需项目对照 |
| 权益产量 | 按公司持股比例折算属于紫金的产量，不一定等于矿山100%总产量 | 归母利润更接近权益产量而非项目总产量 |
| AISC | 黄金“全维持成本”，除采矿现金成本外还含维持现有产量所需的部分资本开支等 | 不同公司定义和副产品抵扣不同，不能直接拿单位销售成本硬比 |
| ROIC | 投入资本回报率，近似看项目税后经营利润能否覆盖为项目投入的债务和股权资本 | 判断高价并购买矿是否真正创造价值 |
| TTM | 最近连续12个月 | 比只看一个季度年化更稳健 |
| FCF Yield | 自由现金流/市值 | 看股价对应多少可自由分配现金；本报告代理值不含并购现金支出 |
| 资本成本 | 债权人和股东愿意提供资金所要求的最低回报，包含利息和股东承担风险所要求的回报 | 项目利润若低于这个门槛，即使产量和收入增长，也可能损害每股价值 |
| 经济租 | 公司凭稀缺资源或难复制能力，在支付正常成本和资本回报后仍能长期多赚的部分 | 仅有高毛利不等于有经济租；必须证明竞对难复制且超额回报没有在收购价中被提前付掉 |
| 边界品位 | 在给定价格、成本和技术下，矿石刚好值得开采的最低品位 | 金属涨价会让更多低品位矿看起来可采，所以资源量增加不一定代表技术或竞争力突然变强 |
| 权益金 | 矿企按销售额、产量或合同约定向资源所有者、政府等支付的费用，英文常写 royalty | 它会随产量或价格分走矿山利润，不能只看采矿现金成本 |
