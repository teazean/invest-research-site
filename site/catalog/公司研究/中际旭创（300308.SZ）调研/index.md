# 中际旭创（300308.SZ）
> 数据核验日：2026-07-14

## 速览

中际旭创是全球数据中心高速光模块的主要供应商之一。这个定位来自公司“全球主力供应商”的自述和已披露业务规模，不是独立市占率排名，精确名次仍待外部验证。AI 算力集群扩大、网络从 400G/800G 向 1.6T 升级，使公司在 2024-2026Q1 同时出现收入、毛利率和现金流的跃升，说明产业趋势已经真实进入报表，而不只是概念。公司质量初判为“值得持续深挖”，管理层初判为“较可靠，但不能无条件信任”。股票层面则不同：按 2026-07-13 的非官方收盘价快照计算，约 1.23 万亿元市值、82 倍 TTM PE，已经计入很强的持续增长预期，不能把“好公司”直接等同于“当前价格便宜”。
